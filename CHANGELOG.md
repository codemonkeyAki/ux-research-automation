# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-06

### Added
- Initial release of UX Research Automation
- Jira integration for research task management
- Confluence integration for findings publishing
- Claude AI integration for intelligent analysis
- Pexels integration for competitor images
- PDF report generation
- Email delivery system
- GitHub Actions webhook automation
- Docker containerization
- Helper shell scripts for deployment
- Comprehensive logging system
- TypeScript support
- Unit test infrastructure

### Features
- Auto-trigger workflow from Jira issues
- Intelligent competitor analysis
- User insight generation
- Actionable recommendations
- Multi-format report support
- Automated email distribution

### Security
- API key encryption in environment variables
- Secure Gmail app password authentication
- Jira/Confluence token-based auth
